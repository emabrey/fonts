FONTLOG for the Inconsolata and Inconsolata Bold font

This file provides detailed information on both the Inconsolata and Inconsolata Bold font software. This information should be distributed along with the font software and any derivative works.

Basic Font Information

Inconsolata is a monospace font with special attention to detail for high resolution rendering that is designed for printed code listings.

Inconsolata Bold is a Unicode typeface family that supports languages that use the Latin script and its variants, and could be expanded to support other scripts. Specifically, Inconsolata Bold supports the following Unicode ranges: Latin-1.
